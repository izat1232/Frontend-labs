# Task 2 - UserProfile with useEffect

Компонент для загрузки данных пользователя с API. Использует useEffect, loading state, error handling и cleanup.

## Как запустить
1. `npm install`
2. `npm start`

## Функции
- Загрузка данных с JSONPlaceholder API
- Состояние загрузки
- Обработка ошибок
- Кнопка обновления
- AbortController для cleanup
