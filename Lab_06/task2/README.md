# Task 2 - Dynamic Routes

Расширенное приложение с динамическими маршрутами /courses/:id и query параметрами.

## Как запустить
1. `npm install`
2. `npm run dev`

## Функции
- `/courses/:id` - детали курса
- `?sort=asc|desc` - сортировка курсов
- `useParams` - получение ID из URL
- `useSearchParams` - работа с query параметрами
- `useLoaderData` - загрузка данных
