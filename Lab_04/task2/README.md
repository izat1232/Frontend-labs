# Task 2 - Article Manager Refactoring

Рефакторинг монолитного компонента на маленькие: AddArticle, ArticleList, ArticleItem.

## Как запустить
1. `npm install`
2. `npm start`

## Компоненты
- `ArticleManager` - управляет состоянием статей
- `AddArticle` - форма добавления статьи
- `ArticleList` - список статей
- `ArticleItem` - одна статья с toggle
