# Task 1 - Lazy Loading

Приложение с ленивой загрузкой страниц. Использует React.lazy() и Suspense.

## Как запустить
1. `npm install`
2. `npm run dev`

## Страницы (lazy loaded)
- `/dashboard` - Dashboard
- `/settings` - Settings
- `/profile` - Profile

## Компоненты
- `LoadingSpinner` - fallback UI
- `Suspense` - обертка для lazy компонентов
