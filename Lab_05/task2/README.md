# Task 2 - SearchApp

Поиск пользователей с типизированными state и event handlers.

## Как запустить
1. `npm install`
2. `npm run dev`

## Типы
- `User` - интерфейс пользователя
- `React.ChangeEvent<HTMLInputElement>` - тип для onChange
- `React.MouseEvent<HTMLButtonElement>` - тип для onClick

## Функции
- Фильтрация по имени
- Кнопка очистки
- Сообщение "No results found"
