# Task 2 - Custom JSX Elements

Компоненты Card, ProductList и Section. Демонстрация props, children и fragments.

## Как запустить
1. `npm install`
2. `npm start`

## Компоненты
- `Card` - карточка с title и children
- `ProductList` - список продуктов через map
- `Section` - секция с title и children через fragment
