# Task 1 - Typed Components

TypeScript проект с типизированными компонентами UserCard и SkillList.

## Как запустить
1. `npm install`
2. `npm run dev`

## Типы
- `User` - интерфейс для пользователя
- `SkillLevel` - union type для уровня навыка
- `Skill` - интерфейс для навыка

## Компоненты
- `UserCard` - карточка пользователя с optional prop isActive
- `SkillList` - список навыков
