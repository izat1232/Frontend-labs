# Task 1 - Registration Form

Форма регистрации с валидацией. Проверяет имя, email и возраст.

## Как запустить
1. `npm install`
2. `npm start`

## Валидация
- Имя: минимум 2 символа
- Email: правильный формат
- Возраст: 18+

## Функции
- Controlled components
- Валидация при вводе
- Предотвращение отправки формы с ошибками
