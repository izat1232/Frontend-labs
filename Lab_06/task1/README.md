# Task 1 - Student Portal

Многостраничное приложение с React Router. Есть навигация, Layout и 404 страница.

## Как запустить
1. `npm install`
2. `npm run dev`

## Страницы
- `/` - Home
- `/courses` - Courses
- `/about` - About
- `*` - 404 Not Found

## Компоненты
- `Layout` - общий layout с nav и footer
- `Link` - навигация без перезагрузки
